# BeKreative Stories Planner

App mobile per generare piani Stories Instagram giornalieri.

## Setup Vercel

1. Importa questo repository su Vercel
2. Aggiungi la variabile d'ambiente:
   - Nome: `ANTHROPIC_API_KEY`
   - Valore: la tua API key Anthropic
3. Deploy

## Utilizzo

- Apri l'app dal link Vercel
- Seleziona stagione e focus
- Aggiungi nota del giorno (opzionale)
- Carica foto del prodotto (opzionale, max 3)
- Genera il piano
- Copia per Telegram
